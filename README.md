# Financial Analytics Dashboard

## Project Overview
Developed a Financial Analytics Dashboard in Power BI to analyze sales, profit, gross sales, profit margin, and business performance metrics.

## Tools Used
- Power BI
- DAX Measures
- Power Query

## Key Metrics
- Sales
- Profit
- Gross Sales
- Profit Margin

## Key Insights
- Analyzed financial data across multiple countries, products, and business segments.
- Identified sales trends and profitable products.
- Evaluated business performance using interactive visuals and KPIs.
